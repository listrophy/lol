Color Your World

  • $ PS1="\[\e[0;31m\]$\[\e[0m\] "
  • $ PROMPT_FUNCTION=some_function
  • $ PS1="🚀"

Show Your Branch

$ function my_prompt {
    PS1="$(git branch --no-color) "
  }
$ PROMPT_COMMAND=my_prompt

Know Your Status

$ function my_prompt {
  modified=$(
    git status --short |
    grep '^.M'
  )
  PS1="$(
    [[ $modified != '' ]] &&
    echo '~'
  ) "
}
$ PROMPT_COMMAND=my_prompt
(master +~) $ ...

Alias ALL THE THINGS

  • alias ll='ls -lbF'
  • alias la='ls -abF'
  • alias ..='cd ..'
  • alias ::='cd ../..'
  • alias gap='git add -p'
  • alias gc='git commit'
  • alias ducks='du -cks * | sort -rn | head'

Employ Functions as Needed

function take() {
  mkdir -p $1
  cd $1
}

Tab Complete is Your Friend

  • $ brew install bash-completion
  • $ cd ~/de<TAB>
  • $ scp me@remote-server:~/de<TAB>
